Lesson 51 - Getters and Setters

This lesson introduces getters and setters, methods used to control access to an object’s data.
They are a key part of encapsulation, helping protect fields and enforce rules when reading or modifying values.

*Topics Covered*
- What getters and setters are
- Making fields readable with getters
- Making fields writable with setters
- Encapsulation and data protection
- Adding validation logic when setting values

*Notes*
- Getters return the value of a field, while setters update it.
- Fields are typically marked as private and accessed through public getter/setter methods.
- Setters can include validation (e.g., preventing negative prices).
- This approach improves data security, maintainability, and control over how data is used.
