Lesson 55 - ArrayLists

This lesson introduces ArrayLists in Java, a part of the Collection Framework that provides a resizable array for storing objects.
Unlike traditional arrays, ArrayLists can grow or shrink dynamically as elements are added or removed.

*Topics Covered*
- Creating an ArrayList using ArrayList<>()
- Adding elements with .add()
- Accessing elements with .get()
- Modifying elements with .set()
- Removing elements with .remove()
- Getting the size with .size()
- Sorting elements using Collections.sort()
- Iterating through an ArrayList with loops
- Accepting user input to dynamically populate an ArrayList

*Notes*
- ArrayLists can only store objects, not primitive types (use wrapper classes like Integer).
- They are more flexible than arrays because their size is not fixed.
- The Collections class provides useful utility methods like sorting.
- ArrayLists are widely used when the number of elements may change during runtime.
